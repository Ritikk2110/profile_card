Your profile card includes:

The card is centered on the page with a clean and modern look.

The typography is neat, with different font sizes for the name, role, and batch year.

🌐 Social Media Integration

✅ LinkedIn
✅ Facebook
✅ Twitter (X)

These icons are circular, well-spaced, and responsive.

🎨 Hover Effects & Interactivity
Icons "pop out" when hovered, making them feel interactive.
A soft box-shadow effect adds depth when hovering.
Clicking on any icon opens the social media profile in a new tab (background opening optimization is still being adjusted).
